What?

The demo application serves the purpose to

  • showcase the Play Pagelets Module and it's advantages
  • demonstrate how to build a pagelet based application
  • serve as a template for pagelet based projects

Run

Open the terminal and navigate to the play-pagelets-seed folder. Then enter 

activator run
to compile and start the application. Then point your browser to http://localhost:9000 to see the app in action. Run the tests with 
activator test

Details

To find out more, have a look at the README file. The source code and unit tests should give you a good idea how to use Play Pagelets. If you really want learn all the details, check out the Pagelets Module: https://github.com/splink/pagelets
